Express RESTful API
===================

## Dependency

- MongoDB

## Setup

```sh
$ git clone git@github.com:shinoburc/ExpressRESTfulAPI.git
$ cd ExpressRESTfulAPI
$ npm install
$ npm start
```

Access [http://localhost:3000/api/foods](http://localhost:3000/api/foods])
